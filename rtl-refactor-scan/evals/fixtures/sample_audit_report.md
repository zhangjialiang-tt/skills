# RTL 成熟度审计报告 — sample_module

示例模块，用于 smoke runner 校验输出契约。

按五大维度扫描：可综合性、可实现性、时序优化、仿真上板一致性、EDA LA 友好性。

## 七、风险列表

### ⚠️ A 类：必须优先处理

- 混合 blocking/non-blocking 赋值（行 88）：会导致综合结果不确定。

### 🔶 B 类：建议优化

- variable part-select 在 EDA debug 不稳定（行 156）。

### 🟢 C 类：只报告，暂不改

- unpacked array 规模小，暂不拆解（行 300）。

## 九、EDA LA/debug 不稳定写法
